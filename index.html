IntelliBot

A Multimodal AI Assistant for Prompt and System Tasks

About the Project

IntelliBot is an intelligent voice assistant that uses Google's Gemini API to interact with users via speech. It can answer queries, open websites, and respond in real-time, making technology more accessible and intuitive.

Key Features

  • Voice-based login and interaction
  • Real-time answers using Gemini API
  • Speech-to-text and text-to-speech engine
  • Web automation via voice commands
  • GUI with animated assistant display
